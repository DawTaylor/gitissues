## Instructions

### Before you begin

Install `create-react-app`.

```sh
npm install -g create-react-app
```

Clone this repository

```sh
git clone https://github.com/DawTaylor/gitissues.git
```

Install dependencies

```sh
npm install
```

Start dev server

```sh
npm start
```

Go to [localhost:3000](http://localhost:3000)
